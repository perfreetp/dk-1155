# 今日崩溃 - 产品需求文档

## 1. 产品概述

**今日崩溃**是一款轻松温暖的情绪记录小程序，为个人和好友小圈子提供每日的压力与情绪出口。不是严肃的心理咨询，而是帮助你表达、整理和互相接住的情绪陪伴工具。用幽默化解沉重，用记录看见规律，用互助传递温暖。

### 核心定位
- **不做严肃咨询**：只是表达和记录，不提供专业心理治疗
- **轻松幽默**：用自嘲和幽默的方式面对负面情绪，降低心理负担
- **互助社区**：好友小圈子内的匿名分享与温暖回应
- **数据洞察**：帮助用户发现情绪规律，掌握自己的情绪地图

## 2. 核心功能模块

### 2.1 页面结构

| 页面 | 功能定位 | 核心模块 |
|------|----------|----------|
| **今日打卡** | 首页/主要入口 | 崩溃等级选择、触发事件、情绪标签、文字/语音记录、自嘲卡片生成 |

| **设置页** | 个性化配置 | 夜间提醒、隐私设置、数据导出 |

## 3. 详细功能说明

### 3.1 今日打卡页

**核心功能：**
1. **崩溃等级选择**
   - 5个等级：微崩溃（1星）、有点崩（2星）、相当崩溃（3星）、严重崩溃（4星）、彻底崩了（5星）
   - 使用表情图标+文字描述，可视化程度高

2. **触发事件输入**
   - 简短描述什么让你崩溃
   - 支持文字输入（最多500字）
   - 提供常见触发场景快捷标签

3. **情绪标签系统**
   - 预设标签：工作压力、人际关系、家庭矛盾、感情问题、学业负担、健康担忧、经济压力、未来焦虑、无端低落、其他
   - 支持多选，每个记录最多选3个标签

4. **文字/语音记录**
   - 文字：简短的心情描述，支持emoji
   - 语音：录制30秒以内的语音，解放双手
   - 二选一或同时使用

5. **自嘲卡片生成**
   - 基于崩溃等级和触发事件，AI生成一句自嘲语录
   - 示例："今天被甲方爸爸的第18版修改打败了，我的灵魂已经躺平"
   - 可复制分享到社交媒体

6. **隐私设置**
   - 每个记录独立设置可见性
   - 选项：仅自己可见、仅好友可见、公开到互助广场

### 3.2 崩溃日历页

**核心功能：**
1. **日历视图**
   - 月份切换，查看每日情绪状态
   - 颜色编码：灰色（无记录）、浅绿（1-2星）、橙色（3星）、红色（4-5星）

2. **每日摘要**
   - 点击任意日期查看当日情绪记录详情
   - 显示崩溃等级、触发事件、情绪标签

3. **筛选与回顾**
   - 按时间范围筛选（本周、本月、本季度）
   - 按标签类型筛选
   - 按崩溃等级筛选

4. **仅自己可见标记**
   - 隐私记录用锁图标标识
   - 查看他人记录时不可见

### 3.3 互助广场页

**核心功能：**
1. **匿名发布机制**
   - 用户可选择将今日打卡发布到广场
   - 发布时自动匿名，显示崩溃等级（不显示具体内容）
   - 提供安慰回复模板

2. **抱抱机制**
   - 给其他用户的崩溃记录发送"抱抱"
   - 抱抱计数显示在卡片上
   - 收到抱抱时通知发送者

3. **安慰回复收藏**
   - 用户可收藏别人的安慰回复
   - 收藏夹功能，随时查看温暖语录

4. **互动反馈**
   - 对安慰回复点赞
   - 表达"这条安慰到了我"

### 3.4 统计页

**核心功能：**
1. **一周高发时段分析**
   - 统计本周情绪崩溃的高峰时段
   - 可视化时间分布图（折线图/柱状图）
   - 提示"你这周周三晚上最容易崩溃哦"

2. **标签分析**
   - 按标签统计崩溃频率
   - 饼图或柱状图展示
   - 帮助用户发现主要压力来源

3. **月度情绪图导出**
   - 生成月度情绪报告
   - 导出格式：PNG图片
   - 包含当月情绪趋势、主要压力源、安慰语录精选

4. **对比功能**
   - 本月与上月对比
   - 可视化展示情绪变化趋势

### 3.5 设置页

**核心功能：**
1. **夜间提醒设置**
   - 设置提醒时间（如22:00）
   - 开关控制
   - 提醒文案："今天有什么想吐槽的吗？"

2. **隐私管理**
   - 默认可见性设置
   - 历史记录批量修改可见性
   - 数据删除功能

3. **数据导出**
   - 导出月度情绪图（PNG）
   - 导出全部数据（JSON）
   - 备份与恢复

4. **小圈子管理**
   - 创建/加入好友圈子
   - 圈子成员管理
   - 圈子隐私设置

5. **个人中心**
   - 头像、昵称设置
   - 崩溃天数统计
   - 收到的抱抱总数

## 4. 核心交互流程

### 4.1 今日打卡流程

```
打开App → 今日打卡页
  ↓
选择崩溃等级（1-5星）
  ↓
填写触发事件（可选）
  ↓
添加情绪标签（可选）
  ↓
记录心情（文字/语音，二选一）
  ↓
生成自嘲卡片（自动生成，可刷新）
  ↓
设置可见性 → 提交
  ↓
可选：分享到互助广场
```

### 4.2 互助广场交互流程

```
进入互助广场 → 浏览匿名崩溃卡片
  ↓
点击卡片 → 查看详情
  ↓
发送抱抱 / 写安慰回复 / 收藏回复
  ↓
收到他人安慰 → 收藏到我的温暖语录
```

## 5. UI设计风格

### 5.1 整体调性
- **温暖治愈**：不压抑，不过于活泼，保持舒适感
- **幽默自嘲**：用轻松的方式面对负面情绪
- **隐私保护**：明显的可见性标识，让用户掌控自己的数据

### 5.2 色彩方案

**主色调：**
- **温暖珊瑚橙** `#FF8A80` - 代表崩溃但保持温暖
- **治愈薄荷绿** `#81C784` - 代表恢复与平静
- **中性灰紫** `#9575CD` - 用于次要元素和过渡

**情绪等级配色：**
- 1星：浅绿 `#C8E6C9`
- 2星：黄绿 `#DCEDC8`
- 3星：橙色 `#FFCC80`
- 4星：橙红 `#FFAB91`
- 5星：红色 `#EF9A9A`

**背景色：**
- 主背景：米白 `#FAFAFA`
- 卡片背景：纯白 `#FFFFFF`
- 深色强调：深紫 `#5E35B1`

### 5.3 字体选择
- **标题字体**：思源黑体（Noto Sans SC）或系统默认雅黑
- **正文字体**：圆润的系统中文字体
- **强调文字**：使用加粗或不同颜色突出

### 5.4 图标与emoji
- 使用圆角柔和的图标风格
- 情绪图标：😊😐😢😰😭😤
- 动作图标：🤗（抱抱）、⭐（等级）、📅（日历）、📊（统计）

### 5.5 卡片设计
- 圆角卡片（border-radius: 16px）
- 柔和阴影（box-shadow: 0 2px 12px rgba(0,0,0,0.08)）
- 适当的内边距（padding: 16-24px）

### 5.6 动画与交互
- 页面切换：平滑滑动过渡
- 卡片点击：轻微缩放反馈
- 提交成功：温暖的庆祝动效（飘落的小爱心或拥抱图标）
- 加载状态：骨架屏或温暖的loading动画

## 6. 技术要求

### 6.1 响应式设计
- **移动端优先**：主要针对手机使用场景优化
- **触摸友好**：所有可点击元素 ≥ 44px
- **单手操作**：核心操作区域在屏幕下半部分

### 6.2 数据存储
- **本地存储**：使用localStorage存储用户数据
- **隐私优先**：敏感数据不上传，纯本地化
- **导出功能**：支持数据备份和迁移

### 6.3 性能要求
- 首屏加载时间 < 2秒
- 页面切换流畅
- 动画不掉帧
