# 今日崩溃 - 技术架构文档

## 1. 架构设计

```
┌─────────────────────────────────────────────────────────┐
│                    前端应用层 (React)                     │
├─────────────────────────────────────────────────────────┤
│  今日打卡 │ 崩溃日历 │ 互助广场 │ 统计页 │ 设置页        │
├─────────────────────────────────────────────────────────┤
│                    状态管理层 (Context + Hooks)           │
├─────────────────────────────────────────────────────────┤
│                  工具函数层 (Utils)                       │
├─────────────────────────────────────────────────────────┤
│                  本地存储层 (localStorage)                │
└─────────────────────────────────────────────────────────┘
```

**说明：**
- 纯前端应用，无需后端服务器
- 所有数据存储在浏览器本地
- 使用localStorage实现数据持久化
- 使用React Context管理全局状态

## 4. 数据模型

### 4.1 情绪记录 (EmotionRecord)

```typescript
interface EmotionRecord {
  id: string;                    // 唯一ID (UUID)
  date: string;                  // 日期 (YYYY-MM-DD)
  level: 1 | 2 | 3 | 4 | 5;     // 崩溃等级
  trigger: string;               // 触发事件
  tags: string[];                // 情绪标签数组
  content: string;               // 文字内容
  voiceUrl?: string;             // 语音URL (Base64)
  selfDeprecatingCard: string;   // 自嘲卡片内容
  visibility: 'private' | 'friends' | 'public'; // 可见性
  createdAt: string;             // 创建时间
  updatedAt: string;             // 更新时间
}
```

### 4.2 广场帖子 (SquarePost)

```typescript
interface SquarePost {
  id: string;                    // 唯一ID
  recordId: string;             // 对应记录ID

  hugs: number;                  // 抱抱数
  comments: Comment[];           // 安慰回复
  createdAt: string;             // 发布时间
```


### 7.2 数据持久化

```typescript
// 自定义Hook: useLocalStorage
const useLocalStorage = <T>(key: string, initialValue: T) => {
  const [storedValue, setStoredValue] = useState<T>(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      return initialValue;
    }
  });

  const setValue = (value: T | SetStateAction<T>) => {
    try {
      const valueToStore = value instanceof Function ? value(storedValue) : value;
      setStoredValue(valueToStore);
      window.localStorage.setItem(key, JSON.stringify(valueToStore));
    } catch (error) {
      console.error(error);
    }
  };

  return [storedValue, setValue] as const;
};
```

